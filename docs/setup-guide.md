# Setup Guide

## Prerequisites
- Node.js 18+
- npm or yarn
- Netlify CLI (`npm i -g netlify-cli`)

## Steps
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in API keys
3. `cd automations && npm install`
4. Configure Notion integration at https://www.notion.so/my-integrations
5. Configure Slack app at https://api.slack.com/apps
6. Run `scripts/setup.sh` for first-time initialization
