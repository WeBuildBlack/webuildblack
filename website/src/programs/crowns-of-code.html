Crowns of Code: School Coding Program

This program has been sunset. Below is an archive of what Crowns of Code accomplished.

Crowns of Code was our youth education program where we taught programming in middle and high schools. Our students learned to build tech from video games to e-commerce websites. The program was completely free to schools.

Students were introduced to computer science fundamentals through hands-on, project-based learning. The goal was to build confidence, spark curiosity, and create a culturally affirming space where young coders could thrive.
